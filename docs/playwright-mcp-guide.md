# Playwright MCP ガイド

Claude Code から Playwright MCP を使って、localhost:8501 で動作する Streamlit アプリの UI を確認するためのガイド。

---

## 目的

このプロジェクトでは、Playwright MCP を使って以下を目的としている。

- AI テスト観点整理アシスタントの UI 基本動作を Claude Code から直接確認する
- 未入力エラーの表示確認、正常系フローの確認など、手動での視覚確認を補助する
- テストシナリオのプロトタイピングに活用する

MCP の設定は `.mcp.json`（プロジェクトルート）で project-scoped 管理しているため、リポジトリを clone したメンバー全員が同じ設定を使える。

---

## 前提条件

| 条件 | 確認方法 |
|------|----------|
| Node.js 18+ がインストールされていること | `node --version` |
| npx が使えること | `npx --version` |
| Streamlit アプリが起動していること | `streamlit run app/streamlit_app.py` |

### Playwright ブラウザのインストール（初回のみ）

```bash
npx playwright install chromium
```

---

## セットアップ

### 1. Claude Code の再起動

`.mcp.json` を追加後、Claude Code を**再起動**して MCP を読み込ませる。

起動後、`/mcp` コマンドで `playwright` が一覧に表示されていれば有効化されている。

### 2. Streamlit アプリの起動

```bash
# プロジェクトルートで実行
source .venv/bin/activate
streamlit run app/streamlit_app.py
```

`http://localhost:8501` でアプリが起動していることを確認する。

---

## Claude Code からの利用例

Streamlit アプリが起動した状態で、Claude Code のチャットに以下のように指示する。

### 画面を開いてスクリーンショットを撮る

```
localhost:8501 を開いて、現在の画面のスクリーンショットを撮ってください。
```

### 未入力エラーの確認（バリデーション確認）

```
localhost:8501 を開いて、仕様テキストを入力せずに「観点を生成する」ボタンをクリックしてください。
エラーメッセージが表示されるか確認してスクリーンショットを撮ってください。
```

### 正常系フローの確認

```
localhost:8501 を開いて、以下の操作を順に行ってください：
1. テスト対象種別を「API」に選択する
2. 仕様テキストに「ユーザー登録 API。メールアドレスとパスワードを受け取り、重複チェックと登録を行う。」と入力する
3. 「観点を生成する」ボタンをクリックする
4. 結果が表示されたらスクリーンショットを撮る
```

### クリアボタンの確認

```
観点生成後に「クリア」ボタンをクリックして、入力フィールドと結果がリセットされることを確認してください。
```

### 短すぎる入力のバリデーション確認

```
localhost:8501 を開いて、仕様テキストに「短い」と入力して「観点を生成する」をクリックし、
バリデーションエラーが表示されることを確認してください。
```

---

## 主なシナリオ早見表

| シナリオ | 確認ポイント |
|---------|------------|
| 未入力送信 | 「仕様テキストは必須です」などのエラーメッセージ表示 |
| 短すぎる入力 | 文字数不足エラー（20文字未満）の表示 |
| 正常系入力 | 観点リスト・カテゴリ分類・サマリーの表示 |
| クリア操作 | フォームと結果の初期化 |
| Markdown エクスポート | ダウンロードボタンの表示と動作 |

## トラブルシューティング

### `playwright` が /mcp に表示されない

- Claude Code を再起動する
- `.mcp.json` の JSON 構文にエラーがないか確認する (`cat .mcp.json`)

### ブラウザが起動しない

```bash
# Chromium ブラウザを再インストール
npx playwright install chromium
```

### `localhost:8501` に接続できない

- Streamlit が起動しているか確認: `streamlit run app/streamlit_app.py`
- ポートが競合していないか確認: `lsof -i :8501`
